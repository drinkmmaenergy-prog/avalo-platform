## Data Collection and Usage

### Personal Data We Collect

**Identity Verification:**
- Government-issued ID (encrypted)
- Date of birth (age verification)
- Facial recognition data (verification only)
- Identity verification images are encrypted and never publicly displayed

**Location Data:**
- Used only for matching and event safety
- Approximate location for discovery features
- Precise location only when user explicitly shares
- Location history is not stored long-term

**Usage Data:**
- Profile views and interactions
- Chat and call history
- Content preferences and engagement
- Platform feature usage analytics

## How We Use Your Data

### Matching and Discovery
- Connect users based on preferences and location
- Improve AI-powered match suggestions
- Enable local event discovery
- Facilitate meaningful connections

### Safety and Security
- Verify user identity and age (18+ enforcement)
- Detect and prevent fraud and abuse
- Monitor for policy violations
- Enable panic features and safety timers

### Monetization Features
- Process creator earnings and payouts
- Calculate revenue splits (65/35)
- Track token transactions
- Generate earning reports and analytics

## Data Protection Principles

### We DO NOT:
❌ Sell personal data to third parties

❌ Use location data for advertising  
❌ Expose private conversations to advertisers  
❌ Store payment card details on our servers  

### We DO:
✅ Encrypt all sensitive data in transit and at rest  
✅ Use location only for matching and safety

## Identity Verification — PACK 207

**Verification Images Storage:**
Verification images are stored encrypted and never displayed publicly. Avalo does not sell personal data.

**Purpose:**
- Enforce 18+ age requirement
- Protect users from catfishing and impersonation
- Enable safe and authentic interactions
- Comply with platform regulations

**Storage:**
- Verification images are encrypted at rest
- Stored securely in isolated systems
- Never displayed publicly on profiles
- Automatically deleted after verification period

**Third-Party Verification:**
- We may use certified identity verification providers
- Providers process data solely for verification
- No data is retained by third-party services
- All transfers comply with GDPR standards

## Location Privacy

**How Location is Used:**
- **Matching:** Show users within specified distance
- **Events:** Enable local event discovery
- **Safety:** Support panic button and safety features
- **Discovery:** Surface trending profiles nearby

**Location Controls:**
- Users can hide exact location
- Approximate city/region shown by default
- Precise location only shared when explicitly enabled
- Location sharing can be disabled anytime

**Location Retention:**
- Current location updated when app is active
- Historical location data not stored
- GPS data deleted after session ends
- No location tracking when app is closed

## User Rights

### Access and Export
- Request copy of all personal data (GDPR Article 15)
- Export data in machine-readable format
- Review all data categories we store
- Processing time: up to 30 days

### Deletion and Erasure
- Request account deletion anytime (GDPR Article 17)
- Most data deleted within 30 days
- Some data retained for legal compliance:
  - Payment records (7 years)
  - Anti-abuse logs (1 year)
  - Tax documents (per jurisdiction)

### Correction and Rectification
- Update personal information anytime
- Correct inaccurate data
- Request verification data review
- Appeal incorrect identity verification

## Data Sharing

### We Share Data With:

**Service Providers:**
- Identity verification services (encrypted transfer)
- Payment processors (PCI-DSS compliant)
- Cloud infrastructure providers (encrypted storage)
- Customer support tools (limited access)

**Legal Obligations:**
- Law enforcement (with valid legal request)

- National security requirements
- Child safety investigations (mandatory reporting)

**Business Transfers:**
- In case of merger or acquisition
- Sale of business assets
- Bankruptcy proceedings

### We DO NOT Share:
- Private messages with advertisers
- Identity verification images
- Precise location without consent
- Financial data with third parties (except payment processors)

## Children's Privacy

**COPPA Compliance:**
- Avalo is strictly 18+ only
- No users under 18 are permitted
- Identity verification enforces age requirement
- Accounts found to be under 18 are immediately suspended

**If We Discover Underage User:**
1. Immediate account suspension
2. All data deleted within 24 hours
3. No data retention for underage profiles
4. Report to authorities if required by law

## TYPOGRAPHY

### Primary Font Family
**Inter** (Sans-serif)
- Weights: 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold), 800 (ExtraBold)
- Usage: Body text, UI elements, buttons
- Why: Modern, clean, highly legible, professional

### Display Font Family
**Sora** (Sans-serif) 
- Weights: 600 (SemiBold), 700 (Bold), 800 (ExtraBold)
- Usage: Headlines, hero text, large titles
- Why: Distinctive, premium feel, attention-grabbing

### Monospace Font (Optional)
**JetBrains Mono**
- Usage: Code snippets, token amounts, technical elements
- Why: Clear, modern, tech-forward

### Universal Tone Guidelines

**Brand Personality:**
- Premium but accessible
- Confident but not cocky
- Modern but not trendy
- Professional but not corporate
- Friendly but not casual

**Writing Style:**
- Use short sentences (max 20 words)
- Active voice preferred
- Second person ("you") over third person
- Contractions allowed (we're, you'll, don't)
- Emoticons sparingly (only when appropriate)
- No excessive punctuation (!!!, ???)

**Words to Use Often:**
Premium, authentic, control, transparent, verified, safe, respect, value, quality, earn, build, connect, real, genuine

**Words to Avoid:**

## DESIGN ELEMENT GUIDELINES

### Logo Usage
- Minimum size: 32px height (digital), 15mm (print)
- Clear space: Equal to height of logo on all sides
- Never stretch, rotate, or distort
- Use primary color on light backgrounds
- Use white on dark backgrounds

### Iconography Style
- Line-based icons, 2px stroke weight
- Rounded corners (2px radius)
- Minimal, clean design
- Consistent visual weight
- Use primary color or neutral gray

### Photography Guidelines
**For Creators:**
- Lifestyle over studio shots
- Natural lighting preferred
- Authentic expressions
- Diverse representation
- Professional but not corporate
- Aspirational but achievable

**For Platform UI:**
- Clean, modern interfaces
- High contrast for readability
- Generous white space
- Clear visual hierarchy
- Premium aesthetic

### Button Styles
**Primary CTA:**
- Background: Primary Violet (#7B2CBF)
- Text: White
- Border-radius: 8px
- Padding: 12px 24px
- Font: 16px SemiBold

**Secondary CTA:**
- Background: Transparent
- Border: 2px solid Primary Violet
- Text: Primary Violet
- Same sizing as primary

**Hover States:**
- Primary: Darken by 10%
- Secondary: Fill with primary color

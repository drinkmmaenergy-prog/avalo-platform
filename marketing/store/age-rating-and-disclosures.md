## Content Warnings & Safety Features

### Mandatory Warnings in App
1. **Age Restriction:** "18+ only. Age verification required."
2. **User-Generated Content:** "Content created by users is moderated but may not be suitable for all audiences."
3. **Calendar Safety:** "Meetings must be in public places. Sexual services are strictly prohibited."
4. **AI Section:** "AI Companions section is separate from real dating. All AI chats are clearly labeled."

### Safety Features Implemented
✅ **Age Verification**
  - Phone number + SMS verification
  - Live selfie with liveness detection
  - 18+ age estimation via AI
  - Optional bank authorization (€0.00 hold)

✅ **Content Moderation**
  - AI-powered real-time filtering
  - Human moderator review queue
  - User report system
  - Automated blocking of prohibited terms
  - Image analysis for inappropriate content

✅ **Anti-Abuse Measures**
  - Rate limiting (50 messages/hour, 100 swipes/hour)
  - Spam detection algorithms
  - Device and IP blacklisting
  - Honeypot profiles for bot detection
  - Shadow bans for violators

✅ **User Controls**
  - Block and report features
  - Privacy settings (Incognito mode)
  - Location controls (can disable)
  - Photo approval for NSFW content
